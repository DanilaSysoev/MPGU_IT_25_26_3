### Задача 5

Ваша задача исправить уязвимости в предоставленном api (находится по пути **../sql-injection-lab/**).
Критерием проверки является прохождение тестов.

Возможны два варианта решения:

1. Исправление с помощью параметризованных запросов. За это решение дается **5 баллов** (по 1 за каждый тест)

2. Исправление с помощью ORM. За это решение дается **15 баллов** (по 3 за каждый тест). В качестве OMR используйте alembic.
   Ваша задача настоить его и выполнить базовые миграции в т. ч. с посевом данных.

## Дополнительно
Если api по вашему мнению содержит еще какие-то уязвимости - обоснуйте и исправьте их. За каждую такую уязвимость - **2 балла**
