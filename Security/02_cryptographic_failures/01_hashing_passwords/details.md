### –†–æ—Ç–∞—Ü–∏—è pepper (—Å–º–µ–Ω–∞)

1. **–ü—Ä–æ–±–ª–µ–º–∞:** pepper –Ω–µ–ª—å–∑—è –ø—Ä–æ—Å—Ç–æ ¬´–ø–µ—Ä–µ—Å—á–∏—Ç–∞—Ç—å¬ª –¥–ª—è —É–∂–µ —Å–æ—Ö—Ä–∞–Ω—ë–Ω–Ω—ã—Ö —Ö—ç—à–µ–π –±–µ–∑ –∑–Ω–∞–Ω–∏—è –∏—Å—Ö–æ–¥–Ω—ã—Ö –ø–∞—Ä–æ–ª–µ–π.
2. **–†–µ—à–µ–Ω–∏—è:**

   * **–õ–µ–Ω–∏–≤–∞—è (lazy) —Ä–æ—Ç–∞—Ü–∏—è:** –ø—Ä–∏ —Å–ª–µ–¥—É—é—â–µ–º –ª–æ–≥–∏–Ω–µ –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—è, –µ—Å–ª–∏ –ø—Ä–æ–≤–µ—Ä–∫–∞ –ø—Ä–æ—à–ª–∞ —Å –ø–æ–º–æ—â—å—é —Å—Ç–∞—Ä–æ–≥–æ pepper, –ø–µ—Ä–µ—Å—á–∏—Ç–∞—Ç—å —Ö—ç—à —Å –Ω–æ–≤—ã–º pepper –∏ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å. –¢—Ä–µ–±—É–µ—Ç—Å—è —É–º–µ—Ç—å –ø—Ä–æ–≤–µ—Ä—è—Ç—å —Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º —Å—Ç–∞—Ä–æ–≥–æ –∏ –Ω–æ–≤–æ–≥–æ pepper‚Äô–æ–≤.
   * **–ü—Ä–∏–Ω—É–¥–∏—Ç–µ–ª—å–Ω–∞—è –º–∏–≥—Ä–∞—Ü–∏—è:** –≤—ã–Ω—É–¥–∏—Ç—å –≤—Å–µ—Ö —Å–º–µ–Ω–∏—Ç—å –ø–∞—Ä–æ–ª—å (–Ω–∞–ø—Ä–∏–º–µ—Ä, –ø—Ä–∏ —É—Ç–µ—á–∫–µ pepper) ‚Äî —Ä–∞–¥–∏–∫–∞–ª—å–Ω–æ, –Ω–æ —á–∏—Å—Ç–æ.
3. **–•—Ä–∞–Ω–µ–Ω–∏–µ –Ω–µ—Å–∫–æ–ª—å–∫–∏—Ö pepper‚Äô–æ–≤:** –≤ —Å–µ–∫—Ä–µ—Ç-—Ö—Ä–∞–Ω–∏–ª–∏—â–µ –æ—Å—Ç–∞–≤–∏—Ç—å –º–∞—Å—Å–∏–≤ [current_pepper, previous_pepper,...], –ø—Ä–∏ –≤–µ—Ä–∏—Ñ–∏–∫–∞—Ü–∏–∏ –ø—Ä–æ–±–æ–≤–∞—Ç—å –ø–æ –æ—á–µ—Ä–µ–¥–∏, –ø—Ä–∏ —É—Å–ø–µ—à–Ω–æ–π –ø—Ä–æ–≤–µ—Ä–∫–µ –ø–µ—Ä–µ—Å—á–∏—Ç–∞—Ç—å –∏ —Å–æ—Ö—Ä–∞–Ω–∏—Ç—å —Ö—ç—à —Å current_pepper.

### –ü—Ä–æ—Ç–∏–≤–æ–ø–æ–∫–∞–∑–∞–Ω–∏—è / –æ–≥—Ä–∞–Ω–∏—á–µ–Ω–∏—è

* –ï—Å–ª–∏ attacker –ø–æ–ª—É—á–∏–ª –∏ –ë–î, –∏ pepper ‚Äî –∑–∞—â–∏—Ç–∞ —Ç–µ—Ä—è–µ—Ç—Å—è. Pepper –ª–∏—à—å –ø–æ–≤—ã—à–∞–µ—Ç –±–∞—Ä—å–µ—Ä –ø—Ä–∏ –∫–æ–º–ø—Ä–æ–º–µ—Ç–∞—Ü–∏–∏ —Ç–æ–ª—å–∫–æ –ë–î.
* Pepper —É—Å–ª–æ–∂–Ω—è–µ—Ç –æ–ø–µ—Ä–∞—Ü–∏–∏ –≤–æ—Å—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–∏—è/–º–∞—Å—Å–æ–≤–æ–π –º–∏–≥—Ä–∞—Ü–∏–∏, –µ—Å–ª–∏ –Ω–µ—Ç –º–µ—Ö–∞–Ω–∏–∑–º–∞ lazy-rehash.
* Pepper –Ω–µ –∑–∞–º–µ–Ω—è–µ—Ç —Å–æ–ª—å –∏ —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–µ KDF (argon2/bcrypt).

### –ü—Ä–∞–∫—Ç–∏—á–µ—Å–∫–∞—è —Ä–µ–∫–æ–º–µ–Ω–¥–∞—Ü–∏—è

* –ò—Å–ø–æ–ª—å–∑—É–π—Ç–µ —Å–æ–≤—Ä–µ–º–µ–Ω–Ω—ã–π KDF (Argon2/bcrypt) + —Å–æ–ª—å (–ø–æ-–ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∞—è) + –æ–ø—Ü–∏–æ–Ω–∞–ª—å–Ω—ã–π pepper (—Ö—Ä–∞–Ω–∏—Ç—å –≤ —Å–µ–∫—Ä–µ—Ç–Ω–æ–º –º–µ–Ω–µ–¥–∂–µ—Ä–µ).
* –†–µ–∞–ª–∏–∑–∞—Ü–∏—è: –ª—É—á—à–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å **pepper –≤ –≤–∏–¥–µ —Å–µ–∫—Ä–µ—Ç–Ω–æ–≥–æ –∫–ª—é—á–∞ –¥–ª—è HMAC**, –∞ –∑–∞—Ç–µ–º —Ö—Ä–∞–Ω–∏—Ç—å –ø–æ–ª—É—á–∏–≤—à–∏–π—Å—è —Ö–µ—à –∫–∞–∫ –æ–±—ã—á–Ω–æ (–∏–ª–∏ –≤—Å—Ç—Ä–∞–∏–≤–∞—Ç—å pepper –≤ –≤—Ö–æ–¥ KDF).

#### –ü—Ä–∏–º–µ—Ä (python, —É–ø—Ä–æ—â—ë–Ω–Ω–æ):

```python
import os, hmac, hashlib
from passlib.hash import bcrypt

PEPPER = os.environ["PEPPER"]  # —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ Vault/ENV

def hash_password(password, salt):        # salt ‚Äî per-user, —Ö—Ä–∞–Ω–∏—Ç—Å—è –≤ DB
    # –ø–æ–¥–ø–∏—à–µ–º password+salt —Å –ø–æ–º–æ—â—å—é pepper (HMAC), –∑–∞—Ç–µ–º –æ—Ç–¥–∞–¥–∏–º –≤ bcrypt
    msg = (salt + password).encode()
    keyed = hmac.new(PEPPER.encode(), msg, hashlib.sha256).digest()
    return bcrypt.hash(keyed)            # bcrypt —Å–∞–º –≥–µ–Ω–µ—Ä–∏—Ç —Å–æ–ª—å –∏ –≤–æ–∑–≤—Ä–∞—â–∞–µ—Ç —Ñ–æ—Ä–º–∞—Ç–∏—Ä–æ–≤–∞–Ω–Ω—ã–π —Ö—ç—à

def verify_password(stored_hash, password, salt):
    msg = (salt + password).encode()
    keyed = hmac.new(PEPPER.encode(), msg, hashlib.sha256).digest()
    return bcrypt.verify(keyed, stored_hash)
```

### –í–æ–∑–º–æ–∂–Ω—ã–µ –ø–æ–¥—Ö–æ–¥—ã ( –∏ –∏—Ö –ø–ª—é—Å—ã/–º–∏–Ω—É—Å—ã )

   * ‚àí –Ω—É–∂–Ω–æ —É–ø—Ä–∞–≤–ª—è—Ç—å —ç—Ç–æ–π –∫–æ–ª–æ–Ω–∫–æ–π –ø—Ä–∏ –∫–∞–∂–¥–æ–π –æ–ø–µ—Ä–∞—Ü–∏–∏ (—Ä–∏—Å–∫–∏ —Ä–∞—Å—Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏).

3. **–ò—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å `CryptContext` (passlib) –∏–ª–∏ –∞–Ω–∞–ª–æ–≥ ‚Äî –∏ –µ–≥–æ `needs_update`**
